# Smart Screen - 产品愿景

> **层级**: L1 - 战略与愿景（Why）  
> **角色**: 决策者 / 产品负责人  
> **本质**: 方向约束

## 愿景描述

构建「像截屏一样简单、输出专业视频」的智能录屏工具 —— 自动生成具有演示级别品质的视频，显著降低后期编辑门槛。

**一句话定位**：Screen Studio 的开源平替，专为 Mac 用户打造的本地智能录屏工具。

## 核心价值主张

### 对用户的价值
- **自动化增强**：Auto Zoom、Cursor Smoothing、智能高亮，无需手动编辑
- **输出即成片**：多预设导出，适配社媒与课程，录完即可分享
- **零学习门槛**：开箱即用，同时提供可调参数满足进阶用户
- **本地优先**：数据完全本地处理，无需订阅，一次拥有

### 对开发者的价值
- 完全开源，可自由定制和扩展
- 深度优化 Apple Silicon（M1/M2/M3/M4），性能卓越
- 学习 macOS 多媒体开发的最佳实践项目

## 目标用户

| 用户类型 | 核心需求 | 优先级 |
|---------|---------|--------|
| 教学/培训者 | 突出操作步骤、快捷键展示、自动高亮 | P0 |
| 产品演示者/PM/Designer | 画质清晰、流程清晰、快速分享 | P0 |
| 内容创作者 | 竖屏导出、短视频预设、GIF 支持 | P1 |
| 开发者/内部培训 | 高保真系统音频、精确剪辑 | P1 |

## 成功指标

### 用户体验指标
- **录制启动时间**：< 2 秒
- **导出速度**：1080p@30fps 视频导出速度 ≥ 2x 实时
- **CPU 占用**：录制时 CPU 占用 < 30%（M1 及以上）

### 产品质量指标
- **音视频同步**：偏差 < 50ms（不可感知）
- **Auto Zoom 准确率**：热点识别准确率 > 90%
- **崩溃率**：< 0.1%

## 产品原则

1. **简单优先**：默认配置即可产出专业效果，复杂选项隐藏于设置
2. **性能至上**：充分利用 Apple Silicon 硬件加速，录制过程几乎无感
3. **非破坏性**：所有编辑操作保留原始素材，随时可回溯
4. **本地优先**：所有处理在本地完成，保护用户隐私

## 发展路线图

### Phase 1：MVP（核心可用）
- ✅ 录屏引擎：区域/窗口/全屏 + 麦克风
- ✅ 基本导出：MP4/MOV、预设管理
- ✅ 基础 UI：开始/停止、区域选择、音频开关

### Phase 2：体验增强
- ⏳ Cursor Smoothing & Click Highlight
- ⏳ Auto Zoom / Focus（后处理实现）
- ⏳ Timeline 基础剪辑（Trim/Speed）

### Phase 3：进阶功能
- ⏳ Webcam Overlay & PiP
- ⏳ System Audio Capture（虚拟驱动）
- ⏳ 高级导出：GIF、社媒预设、批量导出
